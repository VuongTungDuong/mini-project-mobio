# Dự án học giữa kafka và zookeeper

Tạo .venv

## Pdm để  quản lý phiên bản

```bash
uv add
```

đê cài cách pkg chưa tải

## Run

```bash
python app_journey_builder_api.py
```

## Test

```bash
python -m tests.test_email_post
```

## Chuyển sang py sang c

```bash
python setup.py build_ext --inplace
```
